📊 Data Warehouse para E‑commerce — SQL Server

Este projeto apresenta a criação de um Data Warehouse para análise de vendas de E‑commerce, utilizando SQL Server e modelagem dimensional (Kimball).
O objetivo é aplicar o conhecimento adquirido na linguagem SQL, modelagem de dados, tabelas fato/dimensão e boas práticas de organização.

🧱 Modelagem Dimensional

A Modelagem Dimensional,é uma técnica de organização de dados voltada para análise, não para operações do dia a dia.
Consiste em facilitar consultas rápidas, intuitivas e analíticas, especialmente em Business Intelligence (BI), dashboards e relatórios.
Estruturalmente composta por dois tipos de tabelas:

Tabelas Dimensão → descrevem quem, o quê, onde, quando;

Tabelas Fato → registram eventos numéricos (vendas, acessos, pagamentos).

🗄️ Data Warehouse (DW)

Para estruturação desse projeto aplicando a Modelagem Dimencional, utilizamos o DW.
 
Data Warehouse (DW): é um ambiente de armazenamento de dados.Criado para análise estratégica.
É nesse ambiente que se reúne informações de várias fontes, organiza e padroniza esses dados para que possam ser usados em:

-relatórios,indicadores,tomada de decisão,análises históricas e etc.
